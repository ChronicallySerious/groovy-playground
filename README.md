# groovy-playground

Snippets I created while learning Groovy

# Running a snippet

All snippets use Groovy 3.0.8

```shell
$ groovy --version
Groovy Version: 3.0.8 JVM: 1.8.0_265 Vendor: AdoptOpenJDK OS: Windows 10
```

To run a snippet (e.g. `print.groovy`), simply run:

```
groovy print.groovy
```
